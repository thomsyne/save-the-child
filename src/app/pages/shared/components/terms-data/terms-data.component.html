ANNEXURE A – PLATFORM DESCRIPTION AND SPECIFICATIONS

Product Description

The RexRetail product is a Mobile Point of Sales Solution that allows for small to medium sized merchants digitize their retail operations around inventory management, payments, Store management, Cashier management etc. RexRetail is an android-based application that works on the NEXGO N3 and N5 device as the primary device type due to its robust features and durability.

Product Components

RexRetail comprises of three key components which include:

1. Android Application: The RexRetail application is an android based app that can be installed on the Global Accelerex terminals. The application can also be installed on any android mobile device with minimum OS version of 5.1. The application allows users interact with features such as Sales Management, Customer Management etc.

2. Android POS Device: The RexRetail Mobile POS Solution comes pre-installed in the Global Accelerex Android POS device (N3,N5) or any other device that Global Accelerex decides to introduce from time to time.

3. Web Admin Portal: The RexRetail application is available on the web by accessing the url: https://rexretail.globalaccelerex.com/login. The users can logon to the web securely and this application is also responsive, which means the users can access the web functionality through the web/search engine on their mobile or any other android devices.
